# Code Tour

디버거처럼 코드를 탐색하는 인터랙티브 도구

[English](./README.md)

## Code Tour란?

Code Tour는 코드 실행 흐름을 단계별로 탐색할 수 있는 Claude Code 플러그인입니다. 파일을 직접 찾아다니는 대신, Claude가 코드 경로를 따라 안내해줍니다.

```
┌─────────────────────────────────────────────────┐
│ [2/5] src/auth/AuthService.java:120             │
│ ───────────────────────────────────────────────│
│  119 │   public User validateToken(String tok) {│
│  120 │ →   TokenPayload payload = [a]jwtParser. │
│  121 │         parse(token);                    │
│  122 │     return [b]userRepo.findById(         │
│  123 │         payload.getUserId());            │
│                                                 │
│ JWT 토큰을 파싱하고 DB에서 사용자를 조회합니다.  │
├─────────────────────────────────────────────────┤
│ [1] ← 이전  [2] 다음 →  [3] 드릴다운  [q] 종료  │
└─────────────────────────────────────────────────┘
```

## 설치

```bash
# 마켓플레이스 추가
/plugin marketplace add gnoyes/code-tour

# 플러그인 설치
/plugin install code-tour@gnoyes-code-tour
```

## 빠른 시작

```bash
# 기능으로 투어 시작
/tour 로그인 기능

# 특정 위치에서 시작
/tour src/auth/AuthService.java:45

# 자연어로 요청
결제 흐름을 코드 투어로 보여줘
```

## 기능

### 단계별 탐색
간단한 명령으로 코드 흐름을 따라갑니다:
- `1` 또는 `이전` - 이전 단계
- `2` 또는 `다음` - 다음 단계
- `q` 또는 `종료` - 투어 종료

### 드릴다운
함수 안으로 들어가기:
- 호출되는 함수는 `[a]`, `[b]` 등으로 표시됩니다
- 알파벳이나 함수 이름을 입력해서 들어갈 수 있습니다
- "이전"을 누르면 상위 스코프로 돌아갑니다

### 자연어 질문
투어 중 언제든 질문할 수 있습니다:
- "이 함수는 뭘 반환해?"
- "이 함수를 호출하는 곳은?"
- "여기서 실패하면 어떻게 돼?"

### IDE 연동
`파일:라인` 링크를 클릭하면 연결된 IDE에서 해당 위치가 열립니다.

### 스마트 처리
- **분기점**: if/switch에서 어느 경로를 따라갈지 선택
- **외부 라이브러리**: 문서 보기 또는 건너뛰기 옵션
- **순환 참조**: 감지 및 경고

## 사용 예시

### 기능 탐색
```
> /tour 사용자 인증

사용자 인증 코드 투어를 시작합니다...
[AuthController.login()을 진입점으로 표시]
```

### 구현 후 검토
```
> (기능 구현 완료 후)
> 방금 구현한 거 코드 투어로 보여줘

UserService.createUser()에서 투어를 시작합니다...
```

### 디버깅 조사
```
> /tour NullPointerException 발생하는 함수

가능한 위치 발견: OrderService.java:234
투어를 시작합니다...
```

## 명령어

| 명령어 | 설명 |
|--------|------|
| `/tour <대상>` | 대상에서 투어 시작 (파일:라인, 함수, 또는 기능) |
| `/tour` | 시작점 선택 프롬프트 표시 |
